# 🚗 GuessTheCar — Bot Discord

**GuessTheCar** était un bot Discord développé sur mesure pour proposer un jeu de devinettes automobile. Le projet **a été arrêté** et n’est plus actif ni maintenu. Son code source reste **privé**.

---

## 🎯 Objectif

Le bot permettait aux utilisateurs de Discord de tester leurs connaissances automobiles en devinant des marques et modèles de voitures à travers un jeu interactif et compétitif.  
> Le bot n’est plus disponible pour jouer.

---

## 🕹️ Fonctionnement global

Avant l’arrêt du projet :  
- Les joueurs pouvaient lancer une partie avec la commande `/guesscar`.  
- Le bot affichait le pays et le joueur devait deviner la marque, puis le modèle.  
- Le bot validait les réponses, donnait des indices et attribuait des points.  
- Les joueurs pouvaient consulter leur **classement**, **statistiques**, **collection**, obtenir de l’**aide**, ou **abandonner** une partie.  
- Un système de **classement dynamique** permettait de suivre les meilleurs joueurs.

---
